Step 3: Manual Start Instructions

If backend is not running, follow these steps:

  1. Open PowerShell as Administrator
  2. Navigate to the backend directory:
    cd "d:\Aman\Personal\Wifi Dashboard Project\nestjs-backend"
  3. Install dependencies (if needed):
    npm install
  4. Start the backend:
    npm run start:dev
  5. Wait for "Application is running on: http://localhost:5500"
  6. Refresh this page and click "Recheck Backend"
